Adding Next Actions

As a business opportunity develops, you are able to identify the next steps you'll need to undertake in order to put together a proposal and secure the opportunity as a sale.  As you schedule these next actions in CentraView, they are automatically added to your calendar.  Use the following steps to add a next action:

  1. From your Activities listing, select the New button.

    Note:     You can also schedule an activity using the Calendar function by selecting the specific time or using the Schedule button.

  2. On the Details tab, enter the following information:

    • Next action title

    • Next action detail

    • Any associated entity

    • Any associated individual

    • Any associated opportunity

    • Any related type

    • Any related field

    • Owner of the appointment

    • Start and end time

    • Priority

    • Whether the next action is available for Private or Public viewing

    • Status

    • Additional notes

  3. Select the checkbox for Remind Me and enter a reminder date and time to receive a reminder of this action.

  4. Select the checkbox for Send Email Invitations to send email invitations to all individuals associated with this action.

  5. Select the Attendees tab and designate who will be responsible for the next action.  You may select an entire Group or individual attendees.

    Note:     You can also track response to your email invitation through the Attendees tab.

  6. Select the Resources tab to reserve any resources, for example, a conference room or an overhead projector, that you require for your next action.

  7. Select the Availability tab to check the availability of the participants.

  8. Select the Recurring tab to schedule an ongoing action.  An ongoing action is one that occurs daily, weekly, monthly, yearly or at some other frequency.

  9. Select the Attachments tab to send a file, either from the CentraView file system or from your desktop, along with the action notification.  You can send files as attachments or links.

  10. Select the Save and Close button to save the call or select the Save and New button to save and add another action.
