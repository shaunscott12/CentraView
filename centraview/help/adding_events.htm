Adding Events

When you add an event to CentraView, you get to schedule these events, invite participants, secure any visual or marketing aids, and track how well your event aided in sales generation.  CentraView automatically adds them to your calendar.  Use the following steps to add an event:

Note: Once you have added your event, you can compose an email to all attendees using the View function.

  1. From your Events listing, select the New button.

  2. Enter the following information:

    • Event title

    • Event description

    • Start and end date and time for the promotion

    • Attendees list

    • Maximum number or attendees

    • Moderator name

  3. Select the Attach Files button to add a file, either from the CentraView file system or from your desktop, along with the event invitation.

  4. Select the Add Attendees button to add individuals to the event invitation.

  5. Select the Save and Close button to save the event or select the Save and New button to save and add another event.
