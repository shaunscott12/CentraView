Setting Automatic Email Checking Preferences

CentraView allows you set a time interval for automatic mail checking.  Use the following steps to set your mail interval:

  1. From the Additional menu, select Preferences.

  2. Select Email from the preferences listing.

  3. Select Automatic Mail Check from the Email preferences listing.

  4. Select the interval from the drop down list.

  5. Select the Update button to record your preference.
