Adding Literature Requests

When a document is requested by a customer or vendor, adding it to CentraView allows you to track the progress of the request, ensuring a timely response.  Literature requests may come as a result of marketing efforts or in response to a proposal.  As you schedule requests for company literature, they are automatically added to your calendar.  Use the following steps to add a literature request:

  1. From your Activities listing, select the New button.

    Note:     You can also schedule an activity using the Calendar function by selecting the specific time or using the Schedule button.

  2. On the Details tab, enter the following information:

    • Literature request title

    • Literature request detail

    • Any associated entity

    • Any associated individual

    • Employee assigned to fulfill the request

    • Delivery method

    • Status

    • Due date

  3. Select the Add Literature button to add files to the document listing.

  4. Select the Save and Close button to save the literature request or select the Save and New button to save and add another request.
