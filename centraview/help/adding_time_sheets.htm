Adding Time Sheets

When a time sheet, which is comprised of timeslips, is submitted through Accounting to Human Resources, entering the information into CentraView ensures that it will be processed in a timely fashion.  Use the following steps to add a time sheet:

  1. From your Time Sheets listing, select the New button.

  2. Enter the following information:

    • Employee name

    • Reporting manager

    • Date range for the expense

    • Expense description and any additional notes

  3. To add individual time slips, select the New Slip button and enter the following information:

    • Project name

    • Task name

    • Ticket reference

    • Description of the time slip

    • Date

    • Start and end time

    • Total break time

  4. Select the Permissions tab to determine which employees may view, modify or delete the timeslip.

  5. Select the Save and Close button to save the timeslip or select the Save and New button to save and add another timeslip.

  6. Select the Save and Close button to save the timesheet or select the Save and New button to save and add another timesheet.
