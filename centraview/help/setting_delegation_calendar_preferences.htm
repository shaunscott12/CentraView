Setting Delegation Calendar Preferences

CentraView allows you to specify which employees can schedule activities to your calendar.  Use the following steps to set your delegation calendar preferences:

  1. From the Additional menu, select Preferences.

  2. Select Calendar from the preferences listing.

  3. Select Delegation from the calendar preferences listing.

  4. Use the Search button to list specific employees, if necessary.

  5. Select an employee from the listing on the left of the screen.

  6. Select the Add button to add the employee to your delegation list.

  7. If you want to eliminate an employee's delegation privileges, select the employee name from the right hand of the screen and select the Remove button.

  8. Select the Save button to save your preferences.
