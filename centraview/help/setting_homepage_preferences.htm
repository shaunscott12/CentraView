Setting Homepage Preferences

CentraView allows you to personalize your homepage by setting preferences on the refresh rate and the visible information.  Use the following steps to set your homepage preferences:

  1. From the Additional menu, select Preferences.

  2. Select Home from the preferences listing.

  3. Select the refresh rate in minutes and seconds using the drop down lists.

  4. Highlight a portlet and select the Add button to make the portlet visible on your homepage.

  5. Highlight a portlet and select the Remove button to remove the portlet from your homepage.

  6. Select the Save button to save your preferences.
