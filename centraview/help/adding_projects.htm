Adding Projects

When confronted with a larger undertaking and schedule them as projects, CentraView adds them to your calendar.  Once a project is added, you will want to add tasks to the project.  Tasks are the steps towards the project completion.  Use the following steps to add a project:

Note: Once you have added your project, you can associate additional information, for example, tasks, individuals, notes or payments, using the bottom of the screen presented in the View function.

  1. From your Projects listing, select the New button.

  2. Enter the following information:

    • Project title

    • Project description

    • Any associated entity

    • Project contact

    • Project manager

    • Project team

    • Start and end date for the project

    • Status

    • Number of budgeted hours

  3. Enter information in the custom fields, if necessary.

  4. Select the Save and Close button to save the projector select the Save and New button to save and add another project.
