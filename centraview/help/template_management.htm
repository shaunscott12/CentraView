Template Management

Template Management allows you to create and edit templates for Proposals, Mass Emails, Mass Mailings and Support Ticket Printing.

 

  • Simply click on the title of an existing template to edit it or select "New Template" to create a new template.

  • Give the Template a name and specify its type.

  • Format the template in the html editor provided and save when satisfied.  You can use the Tags below top populate the templates with customer information:

    • <IndividualID/>

    • <EntityID/>

    • <Company/>

    • <FirstName/>

    • <MiddleInitial/>

    • <LastName/>

    • <Title/>

    • <PrimaryAddress/>

    • <Street1/>

    • <Street2/>

    • <City/>

    • <State/>

    • <Zip/>

    • <Country/>

    • <Website/>

    • <ExternalID/>

    • <Source/>

    • <Fax/>

    • <Home/>

    • <Main/>

    • <Mobile/>

    • <Other/>

    • <Pager/>

    • <Work/>

    • <Email/>
