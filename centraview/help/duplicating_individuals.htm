Duplicating Individuals

When entering information on individuals within the same grouping or entity, you may find that much of the information is similar.  Having the ability to duplicate an individual, and make the appropriate changes, can save you time and effort.  Use the following steps to duplicate an individual:

  1. From your Individual listing, select the checkbox next to the individual you'd like to duplicate.

  2. Select the Duplicate button.

  3. A record opens with the individual information populated for your convenience.  Change the individual name and any additional information.

  4. Select the Save and Close button to save and return to the Individual listing or select the Save and New button to save and duplicate another individual.
