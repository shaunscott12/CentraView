Adding Tasks

Project tasks are milestones associated with the successful completion of a project.  As you add tasks to a project, CentraView adds them to your calendar and notifies employees of their new assignments.  Use the following steps to add a task:

  1. From your Activities listing, select the New button.

    Note:     You can also schedule an activity using the Calendar function by selecting the specific time or using the Schedule button.

  2. On the Details tab, enter the following information:

    • Task title

    • Task description

    • Any associated project

    • Any parent tasks

    • If there are task milestones

    • Manager of the task

    • Start date and due date

  3. Select the Add Members button to add the names of the employees to whom the project task is assigned.

  4. Enter information in the custom fields, if necessary.

  5. Select the Permissions tab to determine which employees may view, modify or delete the task.

  6. Select the Save and Close button to save the task select the Save and New button to save and add another task.
