About Project Tasks

Project tasks are milestones associated with the successful completion of a project.  Managing tasks and remembering deadlines can require quite a bit of effort, but with CentraView, all your important information is accessible in one spot.  The Project task function allows you to add tasks to your project, notify important participants, remind yourself of upcoming milestones, track task progress and even submit timesheets to Accounting.

With CentraView, you can

  • Add project tasks

  • View project tasks

  • Delete project tasks

  • Duplicate project tasks

For an example of how project tasks work with the rest of the CentraView functionality, see How do I use CentraView?
