Administering Attic Data

Working in tandem with the Garbage distinction is the Attic.  Any user can delete information from the system.  Deleted information is relocated to Garbage, where it can be permanently deleted, restored to its original location or sent to the Attic for safekeeping.  Only administrators can view, delete, move or restore information in the Attic area, safeguarding your system data.  Use the following steps to use the Attic function:

  1. From the Additional menu, select Administration.

  2. Select Data Administration from the administration listing.

  3. Select Attic from the data administration listing.

  4. Select the checkbox next to the appropriate file(s).

  5. Select the Delete button to permanently delete the file.

  6. Select the Restore button to restore the information to its original location.
