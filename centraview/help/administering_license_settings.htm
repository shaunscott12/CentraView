Administering License Settings

When you install a new instance of CentraView, a license is created for your use.  Licenses vary by the number of concurrent users or CentraView and the length of the software contract.  When an upgrade is purchased, you will be required to verify your license.  Use the following steps to administer your license settings:

  1. From the Additional menu, select Administration.

  2. Select Configuration from the administration listing.

  3. Select License Settings from the configuration listing.

  4. Click on the Verify button and wait until your license is updated.

  5. Check the expiration date and number of users to verify your license has been updated.

  6. Select the Save button to verify the new license information.
