Setting up Available Literature

CentraView allows you to customize the list of literature items that can be used to fulfill a literature request for marketing purpose.  Use the following steps to customize the literature request items:

  1. From the Additional menu, select Administration.

  2. Select Module Settings from the administration listing.

  3. Select Marketing from the module listing.

  4. Select Literature from the module settings listing.

  5. Select the Add New button.

  6. Enter the name of the literature item.

  7. Select the Browse button to locate the associated literature file.

  8. Select Save button to save the literature item to the listing.
