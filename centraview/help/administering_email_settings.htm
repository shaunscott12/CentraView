Administering Email Settings

In order for CentraView to send emails such as Activity Invitations or Support Auto-responses you must configure the Global Email Settings.  This tells CentraView what email server to use to send system emails.

  • Admin Email Address

  • Filesystem Storage Path

  • Default Time Zone

  • SMTP Server

  • SMTP Port

  • Login

  • Password

  • Outgoing Mail Server

Email Templates must be configured to populated the automatic emails sent by the system.  This is done by selecting a template from the list, customizing the Subject and Body of the email then clicking on the save button.
