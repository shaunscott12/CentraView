Administering Modules

CentraView allows you to administer personalized setting for each of the system modules, for example, custom views, custom fields and terms.  The following modules have customizable information:

  • Contacts
    - Default Views
    - Custom Views
    - Custom Fields
    - Sources

  • Calendar
    - Working Hours

  • Activities
    - Default Views
    - Custom Views
    - Resources

  • Notes
    - Default Views
    - Custom Views

  • Files
    - Default Views
    - Custom Views

  • Sales
    - Default Views
    - Custom Views
    - Custom Fields
    - Sources
    - Types
    - Stages
    - Terms

  • Marketing
    - Default Views
    - Custom Views
    - Custom Fields
    - Literature

  • Projects
    - Default Views
    - Custom Views
    - Custom Fields
    - Status

  • Support
    - Default Views
    - Custom Views
    - Custom Fields
    - Support Inbox

  • Accounting
    - Default Views
    - Custom Views
    - Custom Fields
    - Terms
    - Invoice Status
    - Order Status
    - Locations

  • HR
    - Default Views
    - Custom Views
    - Custom Fields
    - Suggestion Box

 

Once you have set up your modules, your next step would be to manage your templates.
