Administering Custom Fields

CentraView allows you to determine and maintain custom fields for your specific company needs that are not addressed by the default fields in CentraView.  You can add, delete or duplicate custom fields for any module within CentraView.  Use the following steps to set up custom fields:

Note: Custom Fields can be set for the Contacts, Sales, Marketing, Projects, Support, Accounting and HR modules.

  1. From the Additional menu, select Administration.

  2. Select Module Settings from the administration listing.

  3. Select the module for which you'd like to add custom fields.

  4. Select Custom Fields from the module settings listing.

  5. Select the New Field button.

  6. Enter the field name.

  7. Select the module to which the field will be added.  The default is the module to which you are currently adding fields.

  8. Select the record type from the drop down list, for example, Entity or Individual.

  9. Select whether the field is a Single or Multiple field.

  10. Select the Save & New button to save and add additional fields.

  11. Select the Save and Close button when you are finished adding fields.

 

Once you have set up the your custom fields, your next step would be to manage your templates.
