Administering Global Replace

The Global Replace function allows you to make mass changes to entities, individuals, contacts, or any other system information.  This can be extremely helpful in situations where all accounts from one employee need to be reassigned to another, where there is a name change for a contact, or any other situation where individual changes would be inconvenient and time-consuming.  Use the following steps to globally replace information:

  1. From the Additional menu, select Administration.

  2. Select Data Administration from the administration listing.

  3. Select Global Replace from the data administration listing.

  4. Select Entity or Individual record type.

  5. Select the list to find information in from the drop down list.

  6. Select whether to apply the update to all records or a specific saved search.

  7. Select information from either the Table listing or the Field listing.

  8. Enter the information that will replace the existing data.

  9. Select the Next button to globally replace the information.
