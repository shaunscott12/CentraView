Managing Support Ticket Templates

Support ticket templates ensure that your support staff has valuable, easily recorded and consistent information to work with during the important resolution process.  Use the following steps to set up a support ticket template:

  1. From the Additional menu, select Print Templates.

  2. Select Ticket from the Merge Type listing.

  3. Select whether to use an Entity Saved Search, an Individual Saved Search, or a Specific Entity and/or Individual.

  4. Select the Next button.

  5. Edit the existing default template to fit your company needs, using the formatting buttons to create the look of the template.

  6. Enter a template name.

  7. Select the Save button to save the print template.

 

Once you have set up your templates, your next step would be to manage your contact lists.
