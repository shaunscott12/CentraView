Adding to the Knowledge base

When concepts need larger, more specific explanation, they are added to the knowledge base.  CentraView allows you to add to and categorize your knowledge database.  Use the following steps to add to the knowledge base:

  1. From your Knowledgebase listing, select the New button.

  2. Enter the following information:

    • Knowledge base title

    • Detailed information

    • Category

  3. Select the Permissions tab to determine which employees may view, modify or delete the information.

  4. Select the Save and Close button to save the knowledge base information or select the Save and New button to save and add more information.
