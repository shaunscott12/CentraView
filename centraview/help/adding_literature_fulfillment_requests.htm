Adding a Literature Fulfillment Request

When a potential customer or vendor requests documentation from your company, you should add the request to CentraView.  CentraView adds literature fulfillment requests to your calendar, and you can track their progression using the Literature Request activity status.  Use the following steps to add a request:

  1. From your Literature Fulfillment listing, select the New button.

  2. Enter the following information:

    • Literature fulfillment title

    • Literature fulfillment description

    • Name of requested literature

    • Any associated entity

    • Any associated individual

    • Employee assigned to fulfill the request

    • Delivery method

    • Status

    • Due date

  3. Select the Add Literature button to add files to the document listing.

  4. Select the Permissions tab to determine which employees may view, modify or delete the request.

  5. Select the Save and Close button to save the literature request or select the Save and New button to save and add another request.
