Setting up Locations

CentraView allows you to customize the list of locations on the accounting records.  Use the following steps to customize the available locations:

  1. From the Additional menu, select Administration.

  2. Select Module Settings from the administration listing.

  3. Select Accounting from the module listing.

  4. Select Locations from the module settings listing.

  5. To add a location, enter the location name and select the Add Value button.

  6. To edit a location, select the location name from the listing, enter the modifications and select the Edit Value button.

  7. To remove a location, select the location name from the listing and select the Remove button.

  8. Select Save button to save the locations.
