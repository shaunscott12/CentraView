Setting Default Privileges

CentraView allows you to grant permission to specific individuals for viewing, modifying, and deleting your private information.  Use the following steps to set your default preferences:

  1. From the User Administration menu, select a user.

  2. Select Default Privileges.

  3. Deselect the Public checkbox to allow access to private files.

  4. Highlight an individual and select the Add button for View, Modify or Delete, depending on the privileges you want for the individual.

  5. Highlight an individual and select the Remove button to remove privileges from an individual.

  6. Select the Save & Close button to save your preferences.
