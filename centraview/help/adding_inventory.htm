Adding Inventory

When new items are received into inventory, enter them into CentraView so that are immediately available to your customer base.  Use the following steps to add inventory:

  1. From your Inventory listing, select the New button.

  2. Enter the following information:

    • Item name

    • Item identifier

    • Location

    • Manufacturer name

    • Vendor name

    • Item description

    • Item status

    • Quantity on hand

    • Sold to information

  3. Select the Save and Close button to save the inventory item or select the Save and New button to save and add another inventory item.
