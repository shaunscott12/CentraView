Setting up Default Views

CentraView allows you to administer the default views for any module in the system, selecting which fields will be displayed when a user accesses that module.  These default views can be overridden by the individual user to suit their personal needs.  Use the following steps to set up default views:

Note: Default views can be set for the Contacts, Activities, Notes, Files, Sales, Marketing, Projects, Support, Accounting and HR modules.

  1. From the Additional menu, select Administration.

  2. Select Module Settings from the administration listing.

  3. Select the module for which you'd like to set the default view.

  4. Select Default Views from the module settings listing.

  5. Select the information to appear in the module.

  6. Select the Save button to save the default view.
