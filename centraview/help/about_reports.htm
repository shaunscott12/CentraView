About Reports

With CentraView, you can determine personalized system reports on any information in the database, specifically, contacts, activities, sales, marketing, support, projects, accounting and Human Resources.  Use the following steps to run a report:

  1. From the Additional menu, select Reports.

  2. Select the report category, for example, Contacts, Sales or Marketing, from the reports listing.

  3. Select the report type, for example, Standard, Ad Hoc or Custom, from the reports listing.

    1. Ad-Hoc reports allow you to create customized reports for your company.

    2. Custom Reports allows you to link to an external report available at a specific URL.

  4. Select the ID number of the applicable report or click on New.

  5. Enter dates in the From and To fields to specify a date range. (Not required)

  6. Build your search conditions using the search builder fields.  You may build multiple conditions with the serach builder by adding rows to your characteristics.  Search builder fields include the following:

    • And/Or

    • Record Type, for example, Fields or Projects

    • Field

    • Condition, for example, Begins with or Contains

    • Criteria

      • If searching on a Marketing List, the list must be specified by its list ID, this can be found by looking at the list detail page in Marketing.

      • Dates must be in the following format YYYY-MM-DD

    • Logical Grouping

  7. Select the Run Report button to run your report.

Additionally, you may access the Reports functionality using the icon from the top menu bar.
