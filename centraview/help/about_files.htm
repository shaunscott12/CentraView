About Files

Files supplies you with a powerful file management tool.  Many companies have files spread out across different machines and servers, resulting in directory structures with no central database that become confusing and lead to intense searches for information.  With CentraView, you can determine a centralized, company-wide hierarchy for files, loading important documents from the file folder to your desktop in seconds.

Additionally, CentraView allows you to add descriptive information to each file, explanatory metadata that aids in determining which file is correct for each specific purpose.  Using this powerful centralization, you can also set permissions for each file, relate files back to your entities, individuals or groupings, or make files into News Items that display on the CentraView home page.

 

With CentraView you can

  • Add files

  • View files

  • Delete files

  • Duplicate files

  • Add file folders

For an example of how files work with the rest of the CentraView functionality, see How do I use CentraView?
