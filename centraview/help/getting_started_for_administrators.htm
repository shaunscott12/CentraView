Getting Started for Administrators

CentraView includes powerful tools for the System Administrator, coordinating the set up of user accounts and their security profiles, server and license configuration, data administration functions and the customization of the Centraview homepage.

To lead you towards administering CentraView, this help provides you with the set of steps below, which when followed in sequence provide the quickest, and most efficient, set up possible.

Note: When you install CentraView, a default entity is automatically set up with all of your company information, including employees and employee users.  It is recommended that you change the name of the entity from Default Entity to something representative of your company.  Do this simply by viewing the entity listing and editing the entity name.
