Duplicating Literature Requests

Many of your literature fulfillment requests could contain similar recipients or information.  Having the ability to duplicate a request, and make the appropriate changes, can save you time and effort.  Use the following steps to duplicate a literature request:

  1. From your Activities listing, select the checkbox next to the literature request you'd like to duplicate.

  2. Select the Duplicate button.

  3. A record opens with the literature request information populated for your convenience.  Change the literature request title and any additional information.

  4. Select the Save and Close button to save and return to the Activities listing or select the Save and New button to save and duplicate another literature request.
