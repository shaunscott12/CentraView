Adding Timeslips

As timeslips are submitted to Accounting, it's important that you log them in order to track back to associated project cost.  Timeslips are submitted individually and become part of a larger timesheet.  Use the following steps to add a timeslip to a project:

  1. From your Timeslip listing, select the New button.

  2. Enter the following information:

    • Reference Type

    • Reference

    • Task name

    • Timeslip description

    • Date for timeslip

    • Start and end time for the timeslip

    • Total break time

Note: Tasks must be associated with the project before you can submit a timeslip.

  1. Select the Permissions tab to determine which employees may view, modify or delete the timeslip.

  2. Select the Save and Close button to save the timeslip or select the Save and New button to save and add another timeslip.
