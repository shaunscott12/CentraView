Adding Items

When your business secures new inventory, that is, a new product or service for sale, adding it to CentraView ensures that you can fully track any sales opportunities, proposals or payments for the item.  Use the following steps to add item details:

  1. From your Items listing, select the New button.

  2. Enter the following information:

    • SKU number

    • Item name

    • Item type

    • Item description

    • General ledger account information

    • Tax class

    • Sub-item distinction

    • Manufacturer name

    • Vendor name

    • Item price

    • Item cost

    • Whether this item is linked to inventory

    • Quantity on hand

    • Quantity on order

    • Quantity on backorder

  3. Select the Save and Close button to save the invoice or select the Save and New button to save and add another invoice.
