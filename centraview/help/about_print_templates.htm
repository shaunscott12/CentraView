About Print Templates

CentraView allows you to build and administer print templates, selecting from your own personal templates or using centrally stored company-wide standards.  Using these templates makes it easy and efficient for you to ensure standardized documentation and email presentation.

Note: An email account must be set up before you can use the print templates for email purposes.

The print templates also allow you the capability to merge data, whether for mass mailing or bulk printing.  You can merge data by using a saved search for entities or individuals, or use a specific entity or individual.

Note: Administrators may use the Print Templates functionality to maintain templates for proposals and tickets.

To access the Print Templates functionality, select the icon from the top menu bar.
