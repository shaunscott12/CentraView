Administering Security Profiles

Security profiles allow you to assign user privileges and permissions to all individuals on CentraView.  Some users will need full administrative permissions, while others may be limited to less critical information.  CentraView comes with a default security profile, but it is recommended that you develop profiles which fit the specific needs of your company.  Use the following steps to add a security profile:

  1. From the Additional menu, select Administration.

  2. Select User Administration from the administration listing.

  3. Select Security Profiles from the user administration listing.

  4. Select the New button to add a new profile.

  5. Enter the profile name.

  6. Use the checkboxes to select or deselect applicable permissions.  A checked box designates an included permission.  You can assign permissions for the all of the major informational categories in CentraView.  Each category has an Expand link for more granular profile maintenance.

  7. Select the Save and New button to save and add additional profiles.

  8. Select the Save button when you are finished adding profiles.

 

Once you have established your security profiles, your next step would be to administer user profiles.
