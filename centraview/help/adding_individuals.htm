Adding Individuals

You should have contact information for every person your business deals with at any time.  Every individual is vital to your entity and group listings.  Use the following steps to add an individual:

  1. From the Individual listing, select the New button.

  2. Enter the following information for the individual:

    • First and last name

    • Associated entity

    • Title

    • Full street address, including zip code

    • Country

    • Phone numbers

    • Email address

    • Secondary ID

    • Source

  3. Enter information in the custom fields, if necessary.

  4. Select the Save and Close button to save and return to the Individual listing or select the Save and New button to save and enter another individual.
