Adding File Folders

By using file folders, you can determine a company-wide hierarchy for files, loading from the file folder to your desktop in seconds.  Use the following steps to add file folders to your directory structure:

  1. From your Files listing, select the New Folder button.

  2. Enter the folder name.

  3. Enter the folder name under which to place  the new folder.

    Note:     If the new folder is not a subfolder, place it in the Root directory.

  4. Select whether the folder is for public access or private access only.

  5. Select the Permissions tab to assign file permissions.

  6. Select the Save and Close button to return to your file listing or the Save and New button to create an additional folder.
