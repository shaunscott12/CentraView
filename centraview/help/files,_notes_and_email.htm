How do I Use CentraView? Topics

Adding Files

Keeping vital documentation and business files organized is easy with the CentraView file repository.  Although similar to a file hierarchy you could determine on a server, CentraView allows you to determine this hierarchy without dealing with server permissions or structures, allowing anyone to organize and load important documents from the file folder to a desktop in seconds.

You can build your directory structure by adding directories, or folders, using the New Folder button.

You can add files to the directory structure using the New File button.  After adding files to the directory structure, the file name displays in the file listing.  

When adding files, you can choose to leave them in the root directory or have them automatically placed in any available sub-directory.

Attaching Notes

Whenever you enter a marketing event, proposal, opportunity, or any activity happening within your business, you may also want to enter explanatory notations to help the next person understand something that may not be obvious.

Notes entered through CentraView display whenever a customer or employee requests detailed information on a business activity.  You can easily view all notes you've entered or all notes on one particular subject.

To add notes to any information in CentraView, select the Notes tab and enter a new note.

Using Email

CentraView includes email capabilities, similar to those found in popular email packages such as MS Outlook or Netscape Communicator.

Using your stored contact information, CentraView allows you to easily send invitations, track responses and keep all business participants informed of promotions and events.

Any incoming messages received display on your CentraView home page.

To send or receive email using CentraView, select the Email tab.
