Importing a Marketing List

When you purchase a marketing list from an outside vendor, you will need to import the information into CentraView.  Use the following steps to add a marketing list to your database:

  1. From your List Manager listing, select the New button.

  2. Select whether to add this information to an existing list or create a new marketing list.

  3. Enter a list name and description.

  4. Enter the owner of the list.

  5. Enter a field separator for the list, or select the tab character as your default separator.

  6. Enter a line separator for the list, or select a line return as your default separator.

  7. Select whether you would like a header row to precede the list.

  8. Select the Browse button to select the marketing list to import.

  9. Select the Next button.

  10. Match up the imported fields on the left with the CentraView fields.  You can also enter default values on the far right. (Entity and First or Last Name are required fields.)

  11. Select the Next button to process the marketing list information.

  12. Select the Close button to return to the application and begin using your imported data.
