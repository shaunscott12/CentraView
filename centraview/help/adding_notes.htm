Adding a Note

An explanatory note can be attached to any business activity, proposal, sales opportunity, entity, individual or any information that CentraView contains.  Notes are extremely effective in passing small bits of information onto another CentraView user.  Use the following steps to add a note:

  1. From your Notes listing, select the New button.

  2. On the Details tab, enter the note title.

  3. Enter detailed information for the note.

  4. Enter the entity associated with this note.

  5. Enter the name of the individual associated with this note.

  6. Select the Permissions tab to determine which employees may view, modify or delete the note.

  7. Select the Save and Close button to save the note or select the Save and New button to save the note and add another note.
