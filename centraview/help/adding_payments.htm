Adding Payments

When payments are received for outstanding invoices, log them into CentraView to help keep all accounting issues in close order.  Use the following steps to add payments:

  1. From your Payment listing, select the New button.

  2. Enter the following information:

    • Associated entity

    • Payment amount for invoice

    • Reference information and description

    • Payment method and associated information

  3. Select the Save and Close button to save the payment or select the Save and New button to save and add another payment.
