Duplicating FAQs

Many FAQs can contain similar information.  Having the ability to duplicate an FAQ, and make the appropriate changes, can save you time and effort.  Use the following steps to duplicate an FAQ:

  1. From your FAQ listing, select the checkbox next to the FAQ you'd like to duplicate.

  2. Select the Duplicate button.

  3. A record opens with the FAQ information populated for your convenience.  Change the FAQ title and any additional information, for example, the questions and answers associated with the FAQ.

  4. Select the Save and Close button to save and return to the FAQ listing or select the Save and New button to save and duplicate another FAQ.
