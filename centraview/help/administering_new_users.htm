Administering New Users

One of the most important tasks for the Administrator is setting up new user accounts for the employees of your company.  This enables CentraView to recognize all your relevant employee information.  Use the following steps to set up new users:

  1. From the Additional menu, select Administration.

  2. Select User Administration from the administration listing.

  3. Select Users from the user administration listing.

  4. Select the New button to add a new user.

  5. Select the Lookup button next to the Name field to select an individual this user to which this user will be linked.  (If individual does not already exist, you will need to click on the new individual button to create one.)

  6. Enter the username.

  7. Enter the user password, and confirm the password.

  8. Select Employee, Customer or Administrator from the drop down list, depending on the distinction of the user. (Employees and Administrators must be members of the "Default Entity")

  9. Select the Lookup button to assign a security profile to the user.

  10. Select Yes or No in the Enabled field.  An enabled user registers in the employee listing.

  11. Select the Save and New button to save and add additional users.

  12. Select the Save and Close button when you are finished adding users.

 

Once you have set up your users, your next step would be to set up user email accounts.
