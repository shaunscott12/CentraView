Adding Volume Discounts

Occasionally, in response to sales opportunities and their associated proposals, you may offer volume discounts to your customers.  Use the following steps to add a volume discount:

  1. From your Volume Discount listing, select the New button.

  2. Enter the following information:

    • Discount title

    • Discount description

    • Discount type

    • From and to

    • Discount criteria type

    • Discount Status

  3. Select the Add Items button to add items to the discount.

  4. To remove an item from the discount, select the checkbox next to the item and select the Remove Item(s) button.

  5. Add any necessary notes to the discount.

  6. Select the Save and Close button to save the discount or select the Save and New button to save and add another discount.
