Adding FAQs

CentraView allows you to build a personalized database of frequently asked questions (FAQs) to provide quick, simple answers to the most common inquiries.  As FAQs arise and are answered, use CentraView to add the question and answer to the support database.  Use the following steps to add an FAQ:

  1. From your FAQ listing, select the New button.

  2. Enter the following information:

    • FAQ title

    • Status

  3. Select the Add Question button to add a question and answer for the FAQ.

  4. Select the Save and Close button to save the FAQ or select the Save and New button to save and add another FAQ.
