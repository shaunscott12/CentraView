Setting General Calendar Preferences

CentraView allows you to set general preferences for your calendar, that is, the refresh rate and the default view.  Use the following steps to set your general calendar preferences:

  1. From the Additional menu, select Preferences.

  2. Select Calendar from the preferences listing.

  3. Select General from the calendar preferences listing.

  4. Select your refresh rate in minutes and seconds using the drop down lists.

  5. Select your default view from the available choices.

  6. Select the Save button to save your preferences.
