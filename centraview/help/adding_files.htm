Adding Files

When you add a file to CentraView, you can add descriptive information, explanatory metadata that aids in determining how the file is treated during a search.  You can also set permissions for each file, relate files back to your entities, individuals or groupings, or make files into News Items that display on the CentraView home page.

Before adding files to your directory structure, make sure you create all necessary file folders.  Use the following steps to add a file to your directory structure:

  1. From your Files listing, select the New File button.

  2. Enter the name of the file to be uploaded to the server.

  3. Select the folder into which the file will load.

  4. Enter a title and a brief description for the file.

  5. Enter the name of the file author.

  6. Enter the name of the entity associated with this file, if necessary.

  7. Select a related type from the drop down listing, for example, Opportunity, Proposal, or Project.

  8. Enter a related field, if applicable.

  9. Enter the individual associated with this file, if necessary

  10. Select whether the file is for public access or private access only.

  11. Select whether the file will display in the Customer view.

  12. Select whether the file is a Company News file.  If so, specify the time period to display this file as Company News.

  13. Select the Permissions tab to assign file permissions.

  14. Select the Save and Close button to return to your file listing or the Save and New button to upload an additional file.
