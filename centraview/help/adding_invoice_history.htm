Adding Invoice History

After entering a sales opportunity into the order history, it is time to generate an invoice.  Use the following steps to add invoice information:

  1. From your Invoice History listing, select the New button.

  2. Enter the following information:

    • Order number

    • Billing address

    • Shipping address

    • Invoice status

    • Date of the order

    • Purchase order number

    • Order terms

    • Account Manager name

    • Project name

  3. Select the Add Items button to add items to the invoice.

  4. To remove an item from the invoice, select the checkbox next to the item and select the Remove Item(s) button.

  5. Add any necessary notes to the invoice.

  6. Select the Save and Close button to save the invoice or select the Save and New button to save and add another invoice.
