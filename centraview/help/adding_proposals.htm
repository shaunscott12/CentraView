Adding Proposals

Once you add a sales opportunity, it follows that you would generate proposals tailored to the specific opportunity.  As you schedule and record sales proposals, they are added to your calendar.  Use the following steps to add a proposal:

Note: Once you have added your proposal, you can print, fax or email the proposal as an order form using the Send Now or New Order buttons in the View function.  When viewing the proposal information, you may also determine whether to include this proposal in the sales forecast.

  1. From your Proposal listing, select the New button.

  2. Enter the following information:

    • Opportunity associated with the proposal

    • Proposal title

    • Proposal description

    • Any associated individual

    • Individual billing address

    • Individual shipping address

    • Status

    • Stage

    • Proposal type

    • Probability as a percentage

    • Estimated close date

    • Actual close date

    • Proposal terms

  3. Select the Add Item button to add items to the proposal details.

  4. Enter any special instructions for this proposal.

  5. Select the Attach Files button to add a file, either from the CentraView file system or from your desktop, along with the proposal.

  6. Select the Save and Close button to save the proposal or select the Save and New button to save and add another proposal.
