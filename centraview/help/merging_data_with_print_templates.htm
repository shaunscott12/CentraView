Merging Data with Print Templates

The Mail Merge allows you the capability to merge data, whether for mass mailing or bulk printing.  Use the following steps to use merged data:

  1. From the Marketing menu, select Mail Merge.

  2. Select the merge type from the available choices.

    Note: Selecting Email as the merge type sends a mass email to all applicable recipients.  Selecting Print as the merge type creates a file that can be printed in bulk or saved.

  3. Select whether to use an Entity Saved Search, an Individual Saved Search, or a Specific Entity and/or Individual.

  4. Select the Next button.

  5. Select the template to use from the available templates. (Additional templates can be created or modified by Administrators using Template Management.

  6. Edit your information in the template content area if necessary.

    Note    : You may also change individual items in the body using the supplied formatting tools.

  7. Select the Next button to preview your content.

  8. If the message is correct, select the Next button to merge the data.
