Creating Email Folders

Using the email Folders option, you can create folders and subfolders with names that you assign to sort and store your mail. This allows you to determine a multi-level personal filing system for your mail.  Use the following steps to create a folder and file your mail:

  1. From your Email listing, select the New Folder button.

  2. Enter the Folder Name.

  3. Select the Save and Close button to return to your email or the Save and New button to create an additional folder.
