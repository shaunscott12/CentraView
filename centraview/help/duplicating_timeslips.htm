Duplicating Timeslips

Many timeslips will have similar employee, task or project distinctions.  Having the ability to duplicate a timeslip, and make the appropriate changes, can save you time and effort.  Use the following steps to duplicate a timeslip :

  1. From your Timeslip listing, select the checkbox next to the timeslip you'd like to duplicate.

  2. Select the Duplicate button.

  3. A record opens with the timeslip information populated for your convenience.  Change the timeslip title and any additional information.

  4. Select the Save and Close button to save and return to the Timeslip listing or select the Save and New button to save and duplicate another timeslip.
