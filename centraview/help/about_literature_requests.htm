About Literature Requests

Occasionally, one of your current or prospective customers may need clarifying information about a sales proposal, marketing promotion or a business event.  CentraView helps you keep all the document and recipient information together, making sure the request is filled in an efficient and timely manner.

With CentraView you can

  • Add a request

  • View a request

  • Delete a request

  • Duplicate a request

For an example of how literature requests work with the rest of the CentraView functionality, see How do I use CentraView?
