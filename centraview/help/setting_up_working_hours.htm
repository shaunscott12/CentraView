Setting up Working Hours

CentraView allows you to modify how the calendar displays working hours, allowing you to distinguish between working hours and non-working hours.  Use the following steps to set up the working hours display:

  1. From the Additional menu, select Administration.

  2. Select Module Settings from the administration listing.

  3. Select Calendar from the module listing.

  4. Select Working Hours from the module settings listing.

  5. Select a start and end time which represent the normal working hours for your employees.

  6. Select the days which represent the normal work week days for your employees.

  7. Select Save button to save the working hour distinction.
