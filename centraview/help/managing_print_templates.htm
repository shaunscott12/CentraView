Managing Print Templates

Print templates are valuable towards determining company-wide standards for printed documents, ensuring a consistent professionalism whenever a document is present to a customer.  Use the following steps to set up a print template:

  1. From the Additional menu, select Print Templates.

  2. Select Print from the Merge Type listing.

  3. Select whether to use an Entity Saved Search, an Individual Saved Search, or a Specific Entity and/or Individual.

  4. Select the Next button.

  5. Select the New Template button to create a new template.

  6. Enter the content for the template, using the formatting buttons to create the look of the template.

  7. Enter a template name.

  8. Select the Save button to save the print template.

 

Once you have set up your templates, your next step would be to manage your contact lists.
