Duplicating Files

Use the following steps to duplicate a file:

  1. From your Files listing, select the checkbox next to the file you'd like to duplicate.

  2. Select the Duplicate button.

  3. A record opens with the file information populated for your convenience.  Change the file title and any additional information.

  4. Select the Save and Close button to save and return to the Files listing or select the Save and New button to save and duplicate another file.
