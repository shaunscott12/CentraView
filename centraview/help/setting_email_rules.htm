Setting Email Rules

With CentraView, you can enter specific rules to be applied to you incoming email messages.  These rules could apply to email spam blocks, organizing incoming email into your folder destinations, or any number of uses that you personally determine.  Use the following steps to set email rules:

  1. From your Email listing, select the Rules button.

  2. Select the New Rules button.

  3. Enter the rule name.

  4. Enter a rule description, if necessary.

  5. Select Yes from the Enable drop-down list to enable the rule

  6. Select And (for specific rules) or Or (for less specific rules) from the drop-down box.

  7. Select the Field type from the drop-down box.

  8. Select the Condition type from the drop-down box.

  9. Enter the Criteria for the email address directed by the rule.

  10. Select the Add Rows button and repeat steps 2 through 8 for additional email addresses.

  11. Select an Action to be performed on the email address.

  12. Select the Save and Close button to save the rule or the Save and New button to save the rule and enter an additional rule.
