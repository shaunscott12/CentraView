Adding Expense Forms

Adding expenses to CentraView ensures that the expense payments will be processed in a timely fashion.  Use the following steps to add an expense form:

  1. From your Expense Form listing, select the New button.

  2. Enter the following information:

    • Employee name

    • Reporting manager

    • Date range for the expense

    • Expense description and any additional notes

  3. Select the Add Items button to add expense items to the listing and enter the following information:

    • Quantity

    • Reference type

    • Reference

    • Amount of each item

  4. Select the Save and Close button to save the timeslip or select the Save and New button to save and add another timeslip.
