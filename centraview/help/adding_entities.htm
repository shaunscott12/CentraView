Adding Entities

Since entities are the primary contact record, it is necessary to have entities present in order to add individuals to your contact listings.  Use the following steps to add an entity:

  1. From the Entity listing, select the New button.

  2. Enter the following entity information:

    • Entity name

    • Full street address, including zip code

    • Country

    • Associated website

    • Source

    • Account manager

    • Account team

    • Primary contact name

    • Contact methods and associated numbers

  3. Enter information in the custom fields, if necessary.

  4. Select the Save and Close button to save and return to the Entity listing or select the Save and New button to save and enter another entity.
