Setting up Saved Searches

CentraView includes a powerful search capability, allowing you to search the complete database, returning instances regardless of whether the information resides in contacts, sales, marketing, or anywhere throughout CentraView.

To accomplish this goal, CentraView allows you to use a search builder tool.  With the advanced search builder tool, you can determine which tables, fields or conditions to include in a search.  You can also save search characteristics for later use.  Use the following steps to build a saved search:

  1. From any module, select New Saved Search using the saved search drop down list.

  2. Enter a name for the saved search.

  3. Build your search conditions using the search builder fields.  You may build multiple conditions with the serach builder by adding rows to your characteristics.  Search builder fields include the following:

    • And/Or

    • Record Type, for example, Fields or Projects

    • Field

    • Condition, for example, Begins with or Contains

    • Criteria

      • If searching on a Marketing List, the list must be specified by its list ID, this can be found by looking at the list detail page in Marketing.

      • Dates must be in the following format YYYY-MM-DD

    • Logical Grouping

  4. Select Save and Apply to save the search criteria and run the search, or select Save to simply save the search.

  5. Select Create New to clear the saved search characteristics and build a new search.
