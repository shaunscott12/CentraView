Administering the Customer Logo

CentraView allows you to further customize your homepage by uploading a customer logo, which displays on the left navigation bar.  It is recommended that the graphic for the custom logo be no larger than 146 pixels in width.  Use the following steps to upload the logo:

  1. From the Additional menu, select Administration.

  2. Select Configuration from the administration listing.

  3. Select Customer Logo from the configuration listing.

  4. Select the Browse button to locate your logo.

  5. Confirm using the preview that the logo is correct.

  6. Select the Save button to use this logo.
